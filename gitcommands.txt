
git clone ссылка на репозиторий (копирует репозиторий на наш локальный комп).
touch text2.txt созд документ (только txt json exsl)
nano text2.txt текстовый редактор вносим текст
add text2.txt добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита
commit -m  "название коммита" это команда для записи индексированных изменений в репозиторий Git.
push  отправляем на github
git pull можно воспользоваться чтобы "подтянуть" изменения с удаленнего репозитория в локальный.
