Ветвление.
git checkout -b  название ветки создать и открыть новую ветку.
nano text.txt вносить изменения
git add text.txt добавить содержимое рабочего каталога в индекс
 для последующего коммита
commit -m "название коммита" это команда для записи
 индексированных изменений в репозиторий Git.
git checkout master открыть гл ветку (или вообще люб. ветку).
git merge название ветки которую вливаем
git push -u origin main название ветки отравить на  github
git push отправить на github
