Удалить удаленную и локальную ветку
Нельзя удалить ветку, в которой находитесь.
git branch -d branch_name удалить локальную ветвь,
 если только смержить её с какой-то из веток.
git branch -D branch_name удалить ветку независимо
 от ее статуса push или merge
git push origin --delete branch_name удалить удаленную ветку
